# 🎯 InterviewForge AI – AI-Powered Interview Preparation Platform

InterviewForge AI is a full-stack AI-powered interview preparation platform designed to help students and job seekers prepare for technical and HR interviews effectively.

The platform analyzes resumes, identifies candidate skills, generates personalized interview questions, conducts AI-driven mock interviews, and provides detailed performance feedback reports.

---

## 🚀 Features

### 📄 AI Resume Analysis

* Upload resumes in PDF format
* Extract skills, technologies, and experience automatically
* Generate interview questions based on resume content

### 🤖 Personalized Interview Generation

* Role-specific interview questions
* Technical and HR interview rounds
* Dynamic question generation using AI

### 🎤 Voice-Based Interviews

* Real-time voice interaction
* Natural interview simulation experience
* AI-powered question delivery

### 📊 Performance Reports

* Detailed interview evaluation
* Strength and weakness analysis
* AI-generated feedback and improvement suggestions
* Overall performance scoring

### 🔐 Authentication & Access Control

* Secure user authentication with Firebase
* Protected routes and user-specific data
* Credit-based interview access system

### 💳 Payment Integration

* Razorpay integration for purchasing credits
* Secure payment processing
* Transaction history management

### ⚡ Scalable Backend Architecture

* RESTful API design
* Modular and maintainable code structure
* MongoDB database integration

## 🎯 Purpose

InterviewForge AI was developed to bridge the gap between academic learning and real-world interview preparation.

The platform provides personalized interview experiences by leveraging Artificial Intelligence to analyze candidate profiles and generate relevant interview questions. It helps users identify weaknesses, improve communication skills, and gain confidence before actual interviews.

## 🖥️ Run Locally

```bash
# Clone the repository
git clone https://github.com/pandeyAnjali491/InterviewForge-AI.git

# Open project directory
cd InterviewForge-AI

# Install frontend dependencies
cd client
npm install

# Install backend dependencies
cd server
npm install

# Configure environment variables
Create a .env file and add required API keys

# Start backend server
cd server
npm run dev

# Start frontend
cd client
npm run dev
```
